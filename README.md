## Installation

Install dependencies

```sh
npm install
```

Install iOS dependencies:

```sh
cd ios
pod install
cd ../
```

## Commands

Run iOS app:

```sh
npx react-native run-ios
```

Run Android app:

```sh
npx react-native run-android
```
